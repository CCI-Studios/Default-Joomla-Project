Default Project Layout for Joomla! 2.5

System setup

Make sure the follow are installed for Capistrano deployment:

  • capistrano
  • capistrano-ext for multistage

How to get started

  1. create a fork of the repository, or use it as an remote repo
  2. clone to your local computer
  3. run git submodules initialize --update to setup your repository
  4. begin editing the template and any other needed components
  5. update config/deploy.rb and config/deploy/staging.rb, adding your server account info etc
  6. add your public ssh key to your server and a public ssh key from you server to GitHub

How to deploy

  1. create a database to be used for Joomla on your server
  2. from the command cd into your project directory
  3. type cap deploy:setup to setup the server and install Joomla
  4. type cap deploy to pull the latest version of your project from GitHub to your server
